Una colección de proyectos comunitarios construidos en OpenCode.

:::note
¿Quieres agregar tu proyecto relacionado con OpenCode a esta lista? Envíe un PR.
:::

También puedes consultar [awesome-openchat](https://github.com/awesome-openchat/awesome-openchat) y [openchat.cafe](https://openchat.cafe), una comunidad que agrega el ecosistema y la comunidad.
