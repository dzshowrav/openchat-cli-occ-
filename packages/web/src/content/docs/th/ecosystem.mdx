ชุดโครงการชุมชนที่สร้างขึ้นบน OpenCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OpenCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-openchat](https://github.com/awesome-openchat/awesome-openchat) และ [openchat.cafe](https://openchat.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
