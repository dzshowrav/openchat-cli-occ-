Eine Sammlung von Community-Projekten, die auf OpenCode basieren.

:::note
Möchten Sie Ihr OpenCode-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-openchat](https://github.com/awesome-openchat/awesome-openchat) und [openchat.cafe](https://openchat.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
