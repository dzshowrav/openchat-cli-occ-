Una raccolta di progetti della comunità costruiti su OpenCode.

:::note
Vuoi aggiungere il tuo progetto legato a OpenCode a questa lista? Apri una PR.
:::

Puoi anche dare un'occhiata a [awesome-openchat](https://github.com/awesome-openchat/awesome-openchat) e [openchat.cafe](https://openchat.cafe), una comunità che aggrega ecosistema e community.
