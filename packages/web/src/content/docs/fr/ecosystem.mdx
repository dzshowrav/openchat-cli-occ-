Une collection de projets communautaires construits sur OpenCode.

:::note
Vous souhaitez ajouter votre projet lié à OpenCode à cette liste ? Soumettez une PR.
:::

Vous pouvez également consulter [awesome-openchat](https://github.com/awesome-openchat/awesome-openchat) et [openchat.cafe](https://openchat.cafe), une communauté qui regroupe l'écosystème et la communauté.
